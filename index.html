Eiffie's p5 Playground

This is a place to play with interactive p5 code examples. p5 is processing for javascript which is a widely used graphics package that can run online or off. Each page is a single html file that only references p5.min.js for easy downloading and editing. Grab a copy of p5.min.js here by right clicking and selecting 'save link as'. Similarly each example can be downloaded for offline use or you can grab the whole website from github. Along with learning p5 coding the html files will show some easy ways to setup editable code with javascript. I will be learning along with you and only promise to make the examples unique and hopefully interesting. NOTE: These examples all run without a server. If you need to load assets from a file p5 requires a server.
  • Warped Shelves: Getting started with a static image created in the setup function.
  • String Box: Let's get things moving with the draw function.
  • Orbits: Some things need both setup and draw. Let's try 2 textareas to track the motion of a satellite around a moon, around a planet, around a star.
  • DCT codec: Images can be loaded in p5 but that's too easy. Lets try unpacking a heavily compressed image with the help of a discrete cosine transform.
  • Euler Daisies: You may have heard of Euler Spirals but we will add a slider to create a selectable # of petals.
  • (i*i)%j Tiles: The unpredictability of the Euler Daisies boils down to the interesting behavior of (i*i)%j.
  • Shader Example: Let's edit some glsl and run it with p5's easy to use functions.
  • Sound Example: p5.sound doesn't work offline so we bypass it and go straight to WebAudio. Turns out its easier too!
  • Bouncy the Game: Let's put everything we've learned into a playable game demo.
  • Badges: User input thru html forms can easily be read with p5. Here we will let the user design a name tag.
  • Camera and Filters: Everyone wants to be on camera these days. Digital filters have become a buisness model. Cash in on the craziness by writing your own. Ok maybe bad tatoo face won't make me a billionaire.
  • Accumulating Shader: Using a single offscreen buffer to accumulate ray marching samples on a reflective and refractive fractal.
  • Platonic Solids: Using double buffers to run a simulation limited to p5's 8 bit values.
  • Pinball: Timeout to play a game... but first we have to write it.
