Eiffie Space a testbed for web coding.

Here I will be testing out the functionality of webgl, WebAudio and various related doodads. In the process I will create some games and puzzles to pass the time enjoyably. Let's get started.

  • Rolling Dice is a puzzle where you mark squares in a table by rolling an inked die. The goal is to mark every row and column with unique numbers. We'll learn how to handle events and draw on a canvas.
  • Bouncy is a down-scrolling game where you pilot a space-craft with 2 thrusters. I'll use webgl and WebAudio capabilities and utilize distance estimates, verlet physics and byte-beats!
  • Stitched Stars is an example of webgl frame buffers. 2 off-screen buffers flip positions as texture/buffer to draw pretty patterns.
  • Music Generator creates a musical score based on the Sloth Canon using unicode fonts to draw the score.
  • Coins is a 'simple' puzzle where you match patterns by sliding coins. Now part of Coin Games (first square)
  • Pinball a fast ray-marched pinball game using distance estimates for collision and surface normals with simple physics.
  • DCT Using a discrete Cosine Transform to compress/ decompress an image.
  • Paintball batting a ball that you only see from it's fluid dynamics.
  • Hanoi's Pyramid needs to be moved to higher ground before the river rises.
  • Text Editor Let's make some utilities and learn about handling files on the client. Files are always saved in the download directory specified by the browser. I've added a read aloud button.
  • Base64 Enc/Dec Storing files in code requires some encoding. Base64 offers a bit of compression over decimal encoding. This encoder/decoder will encode files and stringify the result to paste in javascript.
  • Icon Editor We need a slick favicon but most icon editors are point and click. This one is coded.
  • (i*i*n)%j We'll revisit utilities later. Time for some generative pattern fun. You would expect a formula with i*i in it to increase exponentially as i increases but tack on %j and its a crazy world of repeating/reflective patterns used here of drawing tiling and creating music.
  • Music Library We've created alot of music to listen to in the browser but what if you want to download it. Say no more!
  • Shader Editor Some of those music shaders didn't work on my cheap phone. I want to edit them in the browser to see what fails. Here is a simple shader editor.
  • Chesstris Like the name implies its a little bit Chess and a little bit Tetris but mostly just frustrating.
  • Solitare Not Solitare. We couldn't afford the licensing fee.
  • Eval Playpen When console isn't available it only takes a bit of code to implement the most useful feature, log.
  • Sensors Location, location, location. It's always good to know where you stand.
  • Maze Runner Code an AI to navigate a maze.
  • 7 Up Let's use our base64 utility to add images to a coin flipping game. Now part of Coin Games.
  • MIDI Visualizer Connect a MIDI instrument and let's play!
  • MatriX A matrix puzzle. Now part of Coin Games.
  • Angry Bee A 3d sound spacialization example with a vector class, webgl and WebAudio utility functions.
  • Font Table See all the unicode glyphs/codes and load fonts for inspection.
  • Coin Games As long as there are pennies there will be Coin Games. Check back for updated games.
  • Baby IDE This should have been the first project! Let's create an IDE. The last 10% of a project takes 90% of the work so let's leave it out.
  • Dub Recorder Multitrack audio recording the easy way. Dub dub dub!
